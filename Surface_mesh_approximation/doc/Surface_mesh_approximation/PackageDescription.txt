\cgalPkgAuthors{Pierre Alliez, David Cohen-Steiner, Lingjie Zhu}
\cgalPkgDesc{This package implements the Variational Shape Approximation method to approximate an input surface triangle mesh by a simpler surface triangle mesh. The algorithm proceeds by iterative clustering of triangles, the clustering process being seeded randomly, incrementally or hierarchically. While the default function runs an automated version of the algorithm, more control is possible via a class interface.
The API is flexible and can be extended to user-defined error metrics.}
